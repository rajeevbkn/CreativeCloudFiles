An HTML element uses an opening tag, a closing tag and a content (in lower case) in between. All such tags which do not have a pair of opening-closing tags and require only one tag such as 'image tag', 'line-break tag' or 'link tag' etc., should also be closed for a stricter validation. The HTML elements can be nested and empty also at times.

The HTML Attributes are some specific commands written along with an HTML element (within double or single apostophe commas), thus giving them special functionality, character or style. The common types are:-

  1. 'href': used with link-elements
  2. 'src': used with img-elements
  3. 'style': used for in-line styling of html-elements over and above css styles
  4. 'title': gives extra info about the html-element in form of a tool-tip
  5. 'id': gives a unique identification to the element which helps in assigning style and task to it.
  6. 'alt': used in enhancing accessability of an element. Anything defined by it is spoken by the speech recognition software of the broweser for the benefit of visually impaired users.
  7. 'disabled': diables the input element

The headings are important in HTML. The browsers normally searche through headings first. They should be used from h1 type to h6 type in the decreasing order of importance. The headings should not be used to make text bigger and larger as this sometimes produces strange behaviour of browsers. For making text bigger and bolder style attributes should be used invariably.
